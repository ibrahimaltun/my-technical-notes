**Prerequisites**
- Java 17
- Maven 3.6.3
- Gradle 7.5


- Spring Boot uygulaması oluşturmanın en kolay yolu Spring Boot Command Line Interface'dir.
- Spring Boot CLI, Groovy scriptleri çalıştırmamızı sağlayan bir komut satırı aracıdır.

1. [https://docs.spring.io/spring-boot/installing.html](https://docs.spring.io/spring-boot/installing.html)
2. [https://repo.maven.apache.org/maven2/org/springframework/boot/spring-boot-cli/3.3.3/spring-boot-cli-3.3.3-bin.zip](https://repo.maven.apache.org/maven2/org/springframework/boot/spring-boot-cli/3.3.3/spring-boot-cli-3.3.3-bin.zip)

Elle kurulum için aşağıdaki paketlere ihtiyaç vardır.

- **spring-boot-cli-3.3.3-bin.zip**
- **spring-boot-cli-3.3.3-bin.tar.gz**

İndirdikten sonra arşiv dosyasını açın ve install.txt dosyasında verilen adımları izleyin. Herhangi bir ortam kurulumu gerektirmediğini unutmayın.

**main.class**

``` java
package com.example.test;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {
   public static void main(String[] args) {
      SpringApplication.run(DemoApplication.class, args);
   }
   @RequestMapping("/")
	String home() {
		return "Hello World!";
	}
}
```

1. Spring Boot web vağımlıulıkları için pom.xml içerisine aşğıdaki bağımlık satırı eklenir.

``` xml
<dependency>
   <groupId>org.springframework.boot</groupId>
   <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

2. Spring Boot uygulamasını ayağı kaldırdığımızda çalışacak sayfa için **main.class** aşağıdaki gibi güncelleriz.

``` java
import org.springframework.boot.autoconfigure.SpringBootApplication;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
@SpringBootApplication
public class DemoApplication {
	
	public static void main(String [] args) {
		SpringApplication.run(DemoApplication.class, args);
	}
	
	@GetMapping("/")
	String home() {
		return "Welcome to our New Web Page!";
	}
}

```

3. Son adım olarak maven kullanarak projemizi ayağı kaldırırız.

``` bash
$ mvn spring-boot:run
```
